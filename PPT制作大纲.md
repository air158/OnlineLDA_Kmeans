### 【第7页：聚类方法介绍】⭐必需

**内容：**
```
对比算法

KMeans聚类
• 基于距离的划分聚类
• 迭代优化簇中心
• 需要预先指定K值

Single-pass聚类
• 在线增量聚类算法
• 单次遍历即可完成
• 基于相似度阈值

评估指标
1️⃣ 轮廓系数（-1到1，越大越好）
2️⃣ Calinski-Harabasz指数（越大越好）
3️⃣ Davies-Bouldin指数（越小越好）
```

## 🎨 PPT设计建议

### 1. 配色方案

**推荐配色1：商务蓝色系**
- 主色：#3498db（蓝色）
- 辅色1：#e74c3c（红色，用于强调）
- 辅色2：#2ecc71（绿色，用于正面数据）
- 背景：白色或浅灰

**推荐配色2：学术风格**
- 主色：#2c3e50（深蓝灰）
- 辅色1：#e67e22（橙色）
- 辅色2：#27ae60（绿色）
- 背景：白色

### 2. 字体建议

**中文字体：**
- 标题：微软雅黑 Bold / 思源黑体 Bold
- 正文：微软雅黑 / 思源黑体
- 数据：Arial / Helvetica

**字号建议：**
- 标题：32-36pt
- 副标题：24-28pt
- 正文：18-22pt
- 注释：14-16pt

### 3. 排版原则

**6-6原则：**
- 每页不超过6个要点
- 每个要点不超过6个字（中文）或6个词（英文）

**留白：**
- 边距：至少5%
- 元素间距：适当留白
- 不要塞满整页

**对齐：**
- 统一左对齐或居中
- 图表与文字对齐
- 保持一致性

### 4. 图表使用

**必用图表：**
1. `olda_perplexity_analysis.png` - LDA困惑度曲线（第6页）
2. `algorithm_comparison_simple.png` 或 `clustering_comparison.png` - 聚类对比（第8页）
3. `topic_distribution.png` - 主题分布（第10页）

**可选图表：**
4. `kmeans_k_selection.png` - K值选择（第9页）
5. `topic_keywords_top10.png` - 主题关键词（第10页）
6. `data_overview.png` - 数据概况（第4页）

### 5. 动画建议

**进入动画：**
- 标题：淡入
- 要点：从左侧飞入（逐条）
- 图表：淡入

**强调动画：**
- 关键数据：放大/闪烁
- 重要结论：颜色变化

**不要过度：**
- 避免花哨动画
- 保持专业感
