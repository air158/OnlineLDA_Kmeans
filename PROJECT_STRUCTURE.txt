================================================================================

快速开始：

1. 环境配置
   cd /path/to/algo
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

2. 运行实验一（LDA）
   python olda_perplexity_analysis.py
   结果：olda_perplexity_analysis.png

3. 运行实验二（聚类对比）
   python text_clustering_comparison.py
   结果：clustering_comparison.png

4. 查看完整文档
   打开 README.md

实验结果摘要：

【实验一：LDA主题模型】
✓ 最优主题数：4个主题
✓ 最低困惑度：5322.08
✓ 最佳性能：e^(-log(Perplexity)) = 0.000188
✓ 平均训练时间：8.6秒/模型

【实验二：聚类算法对比】
✓ 推荐算法：KMeans (K=3)
✓ 轮廓系数：0.8626 (优秀)
✓ Calinski-Harabasz指数：1224.22
✓ Davies-Bouldin指数：0.2485 (越小越好)
✓ 训练时间：0.15秒
✓ KMeans全面优于Single-pass

【数据统计】
- 文档数量：5,219篇
- 词汇表大小：60,415个词
- 矩阵稀疏度：99.84%
- 主题：日本台湾相关新闻
